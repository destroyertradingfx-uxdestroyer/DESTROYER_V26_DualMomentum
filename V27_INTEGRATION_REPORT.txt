│  ⚠ MEAN-REVERSION EXPECT TO UNDERPERFORM                                     │
│    MBD and SMAD are fundamentally counter-trend strategies. Our prior       │
│    research (SPECTRE, AEGIS, PHOENIX) showed mean-reversion PF < 1.0 on    │
│    EURUSD H4. These may rapidly become unprofitable. Treat as experimental. │
│                                                                              │
│  ⚠ VOLUME FILTER DISABLED                                                    │
│    MBD and SMAD originally used volume spike as confirmation. EURUSD H4     │
│    reports zero volume; filter disabled to allow any signal generation.     │
│    May increase false positives.                                             │
│                                                                              │
│  ⚠ SESSION MAPPING ASSUMPTION                                                 │
│    SRA uses Hour() which returns broker-local time. If your broker uses     │
│    UTC+offset, sessions will be offset. Consider adding InpSRA_OffsetHours. │
│                                                                              │
│  ⚠ SPREAD FILTER: 3.0 pips hardcoded per strategy.                          │
│    May block trades during high-impact news. Consider raising or making    │
│    global if fills are problematic.                                          │
│                                                                              │
